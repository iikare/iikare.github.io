11 MARCH 2021

Due to difficulties with interfacing with the SPI display, the project was changed in order to have different peripherals. The Nunchuk is retained, while the HC-05 Bluetooth module (UART) and a 8x8 LED matrix (I2C) are added. The SD Card and Reader are removed from the project. Now, a maze is generated on the LED matrix, and the game is for the user to complete the maze. The SPI Display is used as a visual timer for maze completion.

4 MARCH 2021

Code implementation for the SPI display was attempted, and the Nunchuk is a work in progress. Implementation of the SD Card Reader and Speaker are also in progress. We may potentially switch to a different SPI display if our current model is deemed too hard to work on.
